<li>

Description

L'élément HTML <li> représente un élément d'une liste. Cet élément doit être contenu dans une liste ordonnée (c'est-à-dire au sein d'un élément <ol>), dans une liste non ordonnée (c'est-à-dire au sein d'un élément <ul>). Pour les listes ordonnées, un indicateur ordinal (un nombre ou une lettre) est utilisé.

Example

<ul>
  <li>50g de sucre</li>
  <li>2 tablettes de chocolat</li>
  <li>1 andouillette de Guéméné</li>
</ul>
